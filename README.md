# Jesse Laguna Portfolio Site 
# 070226

A simple static portfolio website for accounting, AI, data, and workflow-focused job applications.

## Files

- `index.html` — website content
- `styles.css` — website styling
- `assets/Jesse_Laguna_Resume.pdf` — resume PDF
- `assets/Jesse_Laguna_Resume.docx` — resume DOCX

## Quick deploy options

### GitHub Pages
1. Create a new GitHub repository.
2. Upload all files in this folder.
3. Go to Settings > Pages.
4. Set source to the main branch and root folder.
5. Use the generated GitHub Pages URL in job applications.

### Netlify
1. Go to Netlify.
2. Drag and drop this folder.
3. Copy the live site URL.

### Vercel
1. Import this folder or a GitHub repository.
2. Deploy as a static site.
